Accessibility Guidelines

Design system follows WCAG 2.1 AA standards

Color Contrast
All text meets 4.5:1 contrast ratio requirements. Interactive elements have clear focus states with sufficient color contrast.
Keyboard Navigation
All interactive elements are keyboard accessible with visible focus indicators and logical tab order.
Screen Readers
Semantic HTML structure with proper ARIA labels and descriptive text for complex data visualizations.

Brand Guidelines

Consistent application of Cardano and cryptocurrency branding

ADA Symbol Usage
Always use the ₳ symbol before ADA amounts. Apply the cardano-blue color (#0033ad) for brand consistency.
Transaction Hashes
Display transaction hashes in monospace font with truncation (first 6 and last 4 characters) and accent-blue color for links.
Data Formatting
Use consistent number formatting with appropriate decimal places. Large numbers should include thousands separators.
